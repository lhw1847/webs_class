CSS padding

padding

padding 속성은 안쪽 여백을 설정합니다.

속성 설명
padding-top: 10px; 위쪽 안쪽 여백 설정
padding-right: 11px; 오른쪽 안쪽 여백 설정
padding-bottom: 12px; 아래쪽 안쪽 여백 설정
padding-left: 13px; 왼쪽 안쪽 여백 설정
padding: 10px; 위, 오룬쪽 아래, 왼쪽 : 10px
padding: 10px 11px; 위, 아래 : 10px 왼쪽, 오른쪽 11px
padding: 10px 11px 12px; 위 : 10px 왼쪽,오른쪽 : 11px 아래 : 12px
padding: 10px 11px 12px 13px; 위 10px 오른쪽 11px 아래 12px 왼쪽 13px
padding: auto; 자동값, 기본값
