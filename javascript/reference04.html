for()문

fow 문은 조건식을 만족할 때까지 특정 코드를 반복하여 실행한다. 사용 방법은 while 문과 같지만 while 문보다 사용하기 편해 사용 빈도가 높다.

for()문 문법

<script>
    const arr = [100, 200, 300, 400, 500];
    
    //for(초깃값; 조건식; 증감식;)
    for( let i=0; i<arr.length; i++ ){
        document.write(arr[i]);
    }
</script>     //100200300400500

forEach()문

fowEach 문은 배열의 모든 요소를 손쉽게 순회할 수 있도록 해준다. 이 반복문은 루프마다 배열의 각 요소를 지정된 변수에 대입한다. 이렇게 대입받은 변수를 이용하면 루프 안에서 배열의 각 요소에 순차적으로 접근할 수 있다.

forEach()문 문법

<script>
    const num = [100, 200, 300, 400, 500];

    num.forEach(function(el){
        document.write(el);
    });
</script>     //100200300400500
