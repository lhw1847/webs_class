색상

색상은 0부터 360 사이의 값을 가지며, 색상환(color wheel)의 각도를 나타낸다. 색상 값이 0 또는 360이면 빨간색(red)이 되며, 120이면 녹색(green), 240이면 파란색(blue)이 된다.

색을 표현하는 방법

  • 색상 이름으로 표현
  • RGB 색상값으로 표현
  • 16진수 색상값으로 표현

색상 이름으로 표현

<style>
    .blue { color: blue; }
    .green { color: green; }
    .silver { color: silver; }
</style>

RGB 색상값으로 표현

<style>
    .blue { color: #0000FF; }
    .green { color: #008000; }
    .silver { color: #C0C0C0; }
</style>

16진수 색상값으로 표현

<style>
    .blue { color: #0000FF; }
    .green { color: #008000; }
    .silver { color: #C0C0C0; }
</style>
