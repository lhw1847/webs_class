CSS margin

margin

margin 속성은 요소의 바깥쪽 여백을 설정합니다.

문법

margin : length / % / auto

속성 설명
margin-top: 10px; 위쪽 바깥 여백 설정
margin-right: 11px; 오른쪽 바깥 여백 설정
margin-bottom: 12px; 아래쪽 바깥 여백 설정
margin-left: 13px; 왼쪽 바깥 여백 설정
margin: 10px 11px; 위, 아래 : 10px 왼쪽, 오른쪽 11px
margin: 10px 11px 12px; 위 : 10px 왼쪽,오른쪽 : 11px 아래 : 12px
margin: 10px 11px 12px 13px; 위 10px 오른쪽 11px 아래 12px 왼쪽 13px
margin: auto; 자동값, 기본값
margin: 0 auto; 가운데 정렬(블록구조)
