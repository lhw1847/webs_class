속성 설명
<a> <a> 태그는 다른 태그로 이동을 설명해주는 태그입니다.
<abbr> “W3C”, “HTML”, “ASAP” 등과 같이 단어의 축약형(abbreviation)이나 머리글자로만 된 단어(acronym)를 정의할 때 사용합니다.
<acronym> 머리글자로만 된 단어(acronym)를 정의할 때 사용합니다.
<address> 문서나 글의 저자 또는 회사와 연락할 수 있는 정보를 명시할 때 사용합니다.
<applet> 문서에 포함되는 애플릿(웹 페이지에 포함되어 작은 기능을 수행하는 프로그램)을 정의할 때 사용합니다.
<area> 이미지 맵(image-map)에서 하이퍼링크가 연결될 영역을 정의할 때 사용합니다.
<article> 문서, 페이지, 애플리케이션, 또는 사이트 안에서 독립적으로 구분해 배포하거나 재사용할 수 있는 구획을 나타냅니다.
<aside> 페이지의 다른 콘텐츠들과 약간의 연관성을 가지고 있지만, 해당 콘텐츠들로부터 분리시킬 수 있는 콘텐츠로 구성된 페이지 영역을 정의할 때 사용합니다.
<audio> 음악이나 오디오 스트림과 같은 사운드를 정의할 때 사용합니다.
<b> 다른 부가적인 목적 없이 단순히 굵게 표현되는 텍스트를 정의할 때 사용
<base> 해당 문서의 모든 상대 주소(relative URL)에 대한 기본 URL(base URL)과 target 속성값을 정의할 때 사용합니다.
<basefont> 해당 문서의 모든 텍스트를 위한 기본 색상, 크기, 글꼴 집합(font-family) 등을 정의할 때 사용합니다.
<bdi> BDI는 Bi-Directional Isolation을 의미하며, 주위의 텍스트들과는 다른 방향으로 써지는 텍스트 영역을 정의할 때 사용합니다.
<bdo> BDO는 Bi-Directional Override를 의미하며, 자식 요소의 텍스트 방향성(text directionality)을 정의할 때 사용합니다.
<big> 글자 크기가 큰 텍스트를 정의할 때 사용합니다.
<blockquote> 다른 출처로부터 인용된 블록을 정의할 때 사용하며, 이렇게 인용된 블록을 브라우저는 보통 들여쓰기를 사용하여 표현합니다.
<blink> 텍스트를 깜박이게 설정합니다.
<body> <body> 태그는 문서 본문에 해당하는 콘텐츠를 포함하는 영역입니다.
<br> <br> 태그는 텍스트를 줄바꿈 할 때 설정합니다.
<button> 클릭 가능한 버튼을 나타냅니다. 버튼은 양식 내부는 물론 간단한 표준 버튼 기능이 필요한 곳이라면 문서 어디에나 배치할 수 있습니다.
<canvas> 캔버스 스크립팅 API 또는 WebGL API와 함께 사용해 그래픽과 애니메이션을 그릴 수 있습니다.
<caption> <caption> 태그는 table요소의 제목을 설정합니다.
<center> 또다른 블록요소을 포함하거나, 인라인요소(Inline)를 포함할 수 있는 블록속성(block-level) 의 요소이다. 그리고 해당
요소안에 포함된 전체요소는 <center>의 내용안에 가운데 정렬이 된다. (통상적으로 body로 쓰임).
<cite> 저작물의 출처를 표기할 때 사용하며, 제목을 반드시 포함해야 합니다.
<code> <code> 태그는 코드를 나타내는 텍스트를 설정합니다.
<col> <col> 태그는 <colgroup> 요소에 속하는 각 열의 속성을 설정합니다.
<colgroup> <colgroup> 태그는 표의 열을 묶는 그룹을 설정합니다.
<data> 주어진 콘텐츠를 기계가 읽을 수 있는 해석본과 연결합니다.
<datalist> 다른 컨트롤에서 고를 수 있는 가능한, 혹은 추천하는 선택지를 나타내는 option 요소 여럿을 담습니다.
<dd> 용어와 그에 대한 설명을 리스트 형식으로 보여주는 <dl> 요소에서 설명(description) 부분을 정의할 때 사용합니다.
<del> 문서에서 제거된 텍스트의 범위를 나타냅니다.
<details> 사용자가 직접 조작하여 보거나 숨길 수 있는 부가적인 세부사항(additional details)을 명시할 때 사용합니다.
<dfn> 현재 맥락이나 문장에서 정의하고 있는 용어를 나타냅니다
<dialog> 닫을 수 있는 경고, 검사기, 창 등 대화 상자 및 기타 다른 상호작용 가능한 컴포넌트를 나타냅니다.
<dir> 태그는 HTML 리스트(list)를 정의할 때 사용합니다.
<div> <div> 태그는 문서의 영역을 나누거나 그룹화 할 때 설정합니다.
<dl> 요소는 설명 목록을 나타냅니다. <dl>은 dt로 표기한 용어와 dd 요소로 표기한 설명 그룹의 목록을 감싸서 설명 목록을 생성합니다. 주로 용어사전 구현이나 메타데이터(키-값 쌍 목록)를 표시할 때 사용합니다.
<dt> 설명 혹은 정의 리스트에서 용어를 나타냅니다. dl 요소 안에 위치해야 합니다
<em> <em> 태그는 텍스트를 강조할 때 설정합니다.
<embed> dir 전역 특성은 요소의 쓰기 방향을 나타내는 열거형 특성입니다.
<fieldset> <form> 요소에서 연관된 요소들을 하나의 그룹으로 묶을 때 사용합니다
<figcaption> 부모 <figure> 요소가 포함하는 다른 콘텐츠에 대한 설명 혹은 범례를 나타냅니다.
<figure> 독립적인 콘텐츠를 표현합니다. <figcaption> 요소를 사용해 설명을 붙일 수 있습니다. 피규어, 설명, 콘텐츠는 하나의 단위로 참조됩니다.
<font> 텍스트의 폰트(font), 색상 그리고 크기를 정의할 때 사용합니다.
<footer> 가장 가까운 구획 콘텐츠나 구획 루트의 푸터를 나타냅니다. 푸터는 일반적으로 구획의 작성자, 저작권 정보, 관련 문서 등의 내용을 담습니다
<form> 정보를 제출하기 위한 대화형 컨트롤을 포함하는 문서 구획을 나타냅니다.
<frame> 다른 HTML 문서가 표시될 수 있는 특정 영역을 정의하는 element이다. frame은 <frameset>내에서 사용되어야 한다.
<frameset> HTML element로써 <frame> element을 포함하고 사용된다.
<h1> ~ <h6> <h1> ~ <h6> 태그는 섹션의 제목을 설정합니다.
<head> <head> 태그는 제목과 스크립트, 스타일 시트, 메타 정보 등을 제공하는 영역입니다.
<header> <header>는 사이트의 소개 및 메뉴 그룹을 나타내는 섹션을 설정합니다.
<hgroup> 문서 구획의 다단계 제목을 나타냅니다. 다수의 <h1>-<h6> 요소를 묶을 때 사용합니다.
<hr> <hr> 태그는 요소간의 주제별 구분을 설정합니다.
<html> <html> 태그는 웹 문서의 최상위 요소 문서의 루트를 나타냅니다.
<i> 텍스트에서 어떤 이유로 주위와 구분해야 하는 부분을 나타냅니다. 기술 용어, 외국어 구절, 등장인물의 생각 등을 예시로 들 수 있습니다. 보통 기울임꼴로 표시합니다.
<iframe> 중첩 브라우징 맥락을 나타내는 요소로, 현재 문서 안에 다른 HTML 페이지를 삽입합니다.
<img> <img> 태그는 웹 문서에 이미지를 설정합니다.
<input> 웹 기반 양식에서 사용자의 데이터를 받을 수 있는 대화형 컨트롤을 생성합니다.
<ins> 문서에 추가된 텍스트의 범위를 나타냅니다.
<kbd> 키보드 입력, 음성 입력 등 임의의 장치를 사용한 사용자의 입력을 나타냅니다.
<keygen> 공개 키 제출을 용이하게 하기 위해 존재합니다 .
<label> 사용자 인터페이스 항목의 설명을 나타냅니다.
<legend> 부모 <fieldset> 콘텐츠의 설명을 나타냅니다.
<li> <li> 태그는 목록의 항목을 설정합니다.
<link> <link> 외부 파일을 연결할 때 설정합니다.
<main> <main> 태그는 웹 문서의 주요 콘텐츠 영역을 설정합니다.
<map> <area> 요소와 함께 이미지 맵(클릭 가능한 링크 영역)을 정의할 때 사용합니다.
<mark> 현재 맥락에 관련이 깊거나 중요해 표시 또는 하이라이트한 부분을 나타냅니다.
<marquee> 텍스트를 좌우상하로 움직이는 태그이다.
<menu> 일반적으로 텍스트 메뉴, 도구 모음 및 명령 옵션 목록에 사용됩니다.
<menuitem> menu태그로 둘러 쌓인 컨텐츠에 각각의 <menuitem>태그를 배치하여 사용자가 해당 영역에서 마우스 오른쪽 버튼을 클릭하면 <menuitem> 요소들이 메뉴처럼 펼쳐진다.
<meta> html 문서에 대한 모든 정보를 브라우저나 검색엔진에 제공
<meter> 디스크의 사용량, 질문의 결과와 같은 정보를 표시합니다.
<nav> <nav> 는 문서의 부분 중 현재 페이지 내, 또는 다른 페이지로의 링크를 보여주는 구획을 나타냅니다. 자주 쓰이는 예제는 메뉴, 목차, 색인입니다.
<noframes> 프레임이 지원되지 않을 경우 대체 메세지를 표시합니다.
<noscript> 태그 안의 내용은 스크립트를 지원하는 않는 웹브라우저나 스크립트를 실행하지 않도록 설정한 웹브라우저에서만 출력됩니다.
<object> 웹 문서에 포함된 멀티미디어 요소를 정의합니다.
<ol> 순서가 있는 목록형 태그입니다.
<optgroup> <select> 메뉴 <option> 리스트의 그룹화를 설정합니다
<option> <select> 요소의 항목을 정의합니다.
<output> 웹 사이트나 앱에서 계산이나 사용자 행동의 결과를 삽입할 수 있는 컨테이너 요소입니다.
<p> <p> 태그는 단락을 나타낼 때 설정합니다.
<param> <object> 태그의 매개변수를 정의합니다.
<picture> 반응형 사이트와 관련이 있으면 화면 크기에 따라 그에 맞는 이미지를 보여주는 태그입니다.
<pre> <pre> 태그는 HTML에 작성한 내용을 그대로 화면에 표현합니다.
<pregress> 어느 작업의 완료 정도를 나타내며, 주로 진행 표시줄의 형태를 띕니다.
<q> 둘러싼 텍스트가 짧은 인라인 인용문이라는것을 나타냅니다.
<rp> <ruby> 요소를 사용한 루비 주석을 지원하지 않는 경우 보여줄 괄호를 제공할 때 사용합니다.
<rt> 동아시아 문자의 루비 주석에서 발음, 번역 등을 나타내는 텍스트 부분을 지정합니다.
<rtc> <rb> 요소가 표시하는 문자의 의미에 대한 주석을 나타냅니다.
<ruby> 루비 주석을 나타냅니다. 루비 주석은 동아시아 문자의 발음을 표기할 때 사용합니다.
<s> 글자에 취소선, 즉 글자를 가로지르는 선을 그립니다.
<samp> 컴퓨터 프로그램 출력의 예시(혹은 인용문)를 나타냅니다.
<script> 클라이언트 사이드 스크립트와 같은 프로그래밍 코드를 HTML 문서 내에 직접 작성하거나 불러올때 사용한다.
<section> HTML 문서의 독립적인 구획을 나타내며, 더 적합한 의미를 가진 요소가 없을 때 사용합니다.
<select> 옵션 메뉴를 제공하는 컨트롤을 나타냅니다.
<small> 덧붙이는 글이나, 저작권과 법률 표기 등의 작은 텍스트를 나타냅니다.
<source> <audio> 요소나 <video> 요소, <picture> 요소에서 사용할 수 있는 다중 미디어 자원(multiple media resources)을 정의할 때 사용합니다.
<span> 구문 콘텐츠를 위한 통용 인라인 컨테이너로, 본질적으로는 아무것도 나타내지 않습니다
<strike> 텍스트 한가운데 라인을 추가할 때 사용합니다.
<strong> <strong> 태그는 텍스트에 중요성을 강조할 때 사용합니다.
<style> <style> 태그는 웹 문서에 스타일 정보를 설정합니다.
<sub> 아랫첨자(subscript) 텍스트를 표현할 때 사용합니다.
<summary> <details> 요소에 의해 생성되는 대화형 위젯에서 기본적으로 보이는 제목을 정의할 때 사용합니다.
<sup> 윗첨자(superscript) 텍스트를 표현할 때 사용합니다.
<svg> 백터(vector) 이미지를 표현하기 위한 포맷으로 w3c에서 만든 백터 이미지 표준입니다.
<table> <table> 태그는 표를 설정합니다.
<tbody> <tbody> 태그는 표의 본문 영역을 그룹화합니다.
<td> <td> 태그는 표의 열(셀)을 설정합니다.
<template> 추가되거나 복사될 수 있는 HTML 요소들을 정의할 때 사용합니다.
<textarea> 사용자가 여러 줄의 텍스트를 입력할 수 있는 텍스트 입력 영역을 정의할 때 사용합니다.
<tfoot> <tfoot> 태그는 표의 푸터 영역을 그룹화합니다.
<th> <th> 태그는 행이나 열에 타이틀을 설정합니다.
<thead> <thead> 태그는 표의 헤더 영역을 그룹화합니다.
<time> 시간의 특정 지점 또는 구간을 나타냅니다.
<title> <title> 태그는 브라우저의 제목 또는 탭에 표시되는 문서의 제목을 설정합니다.
<tt> 텔레타이프 텍스트(teletype text)를 나타낼 때 사용합니다.
<u> 글자로 표현하지 않는 주석을 가진 것으로 렌더링 해야 하는 텍스트를 나타냅니다.
<ul> 순서가 없는 HTML 리스트(list)를 정의할 때 사용합니다.
<var> 태그는 변수(variable)를 정의할 때 사용합니다.
<video> 비디오 플레이백을 지원하는 미디어 플레이어를 문서에 삽입합니다.
<wbr> 줄바꿈 할 위치를 정의할 때 사용합니다.
