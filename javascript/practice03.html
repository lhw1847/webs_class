결과보기

1.1 for문을 이용해서 테이블을 출력하시오!

1~25까지 숫자를 넣으시오!

{
    let num = 1;
    let table = "<table>";

    for (i = 1; i <= 5; i++) {
        table += "<tr>";
        for (j = 1; j <= 5; j++) {
            table += "<td>" + num + "</td>";
            num++;
        }
        table += "</tr>";
    }
    table += "</table>";

    document.write(table);
}
