속성 설명
grid grid 속성은 하나의 선언에서 명시 적 및 암시 그리드의 모든 속성을 설정합니다.
grid-area grid-area 약식 속성 은 그리드 배치에 선, 범위 또는 아무것도(자동) 기여하여 그리드 영역 의 가장자리를 지정하여 그리드 항목의 크기와 위치 를 지정합니다 .
grid-auto-columns grid-auto-columns 속성은 암시적으로 생성된 그리드 열 트랙 또는 트랙 패턴의 크기를 지정 합니다
grid-auto-flow grid-auto-flow 속성은 자동 배치 알고리즘이 자동 배치 항목을 그리드로 유입 얻을 방법을 정확하게 지정 작동 방식 CSS 속성을 제어합니다.
grid-auto-rows grid-auto-rows 속성은 내재적 생성 그리드 로우의 크기를 지정 트랙 의 트랙 패턴입니다.
grid-column grid-column 약식 속성 은 그리드 배치에 선, 범위 또는 아무것도 제공하지 않음(자동)을 통해 그리드 항목의 크기와 위치 를 지정하고 그리드 영역 의 인라인 시작 및 인라인 끝 가장자리를 지정합니다.
grid-column-end grid-column-end 속성은 그리드 배치에 선, 범위 또는 아무것도 제공하지 않음(자동)으로써 그리드 열 내에서 그리드 항목의 끝 위치를 지정하여 그리드 영역 의 블록 끝 가장자리를 지정합니다.
grid-column-gap grid-column-gap(column-gap) 속성 은 요소의 열 사이 의 간격( 거터 ) 크기를 설정합니다.
grid-column-start grid-column-star 속성은 그리드 배치에 선, 범위 또는 아무것도 제공하지 않음(자동)으로 그리드 열 내에서 그리드 항목의 시작 위치를 지정합니다. 이 시작 위치는 그리드 영역 의 블록 시작 가장자리를 정의합니다.
grid-gap grid-gap(gap) 속성은 행과 열 사이의 간격( 거터 )을 설정 합니다.
grid-row grid-row 약식 속성은 그리드 배치에 선, 범위 또는 없음(자동)을 제공하여 그리드 행 내에서 그리드 항목의 크기와 위치를 지정하여 그리드의 인라인 시작 및 인라인 끝 가장자리를 지정합니다.
grid-row-end grid-row-end 속성은 그리드 배치에 선, 범위 또는 아무 것도 제공하지 않음(자동)으로써 그리드 행 내에서 그리드 항목의 끝 위치를 지정하여 그리드 영역 의 인라인 끝 가장자리를 지정합니다.
grid-row-gap grid-row-gap 속성 은 요소의 그리드 행 사이 의 간격( 거터 ) 크기를 설정합니다.
grid-row-start grid-row-start 속성은 그리드 배치에 선, 범위 또는 아무 것도 제공하지 않음(자동)으로써 그리드 행 내에서 그리드 항목의 시작 위치를 지정하여 그리드 영역 의 인라인 시작 가장자리를 지정합니다.
grid-template grid-template 속성은 약식 속성을 정의하기위한 그리드 열 , 행 및 지역이다.
grid-template-areas grid-template-areas 속성은 그리드 영역의 위치를 ​​지정하는 데 사용되는 4개의 라인 모두를 값으로 사용할 수 있는 속성입니다.
grid-template-columns grid-template-columns 속성 은 행 이름을 정의하고 그리드 열의 크기 조정 기능을 추적합니다.
grid-template-rows 속성 은 행 이름을 정의하고 그리드 행의 크기 조정 기능을 추적합니다.
